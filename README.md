# 四則演算計算コマンド

## 概要
- このコマンドは標準入力から読んだ数値Nに対して四則演算を行うコマンドです。

## 機能概要
- 整数・小数の両方に対応（全ての値はfloatで処理されます。）
- 使用する演算はコマンドライン引数で指定されます。
- minus / div でも自然な演算ができるよう、最初の値を初期値として扱う方式です。

## 使用方法

### インストール(クローン)
```
git clone https://github.com/rinron777/robosys2025-2.git
cd robosys2025-2
```
- 上のコマンドを実行してください。

### 実行方法
```
seq N | ./calc (演算子)
```
#### (N)に任意の数、(演算子)に使用したい機能のコマンドを打ち込んで使用してください。
- 対応コマンド一覧(使用例付属)
  - 加算: `plus`
  - 使用例:(N = 5)
```
$ seq 5 | ./calc plus  
15      #1+2+3+4+5=15
```
  - 減算: `minus`
  - 使用例:(N = 3)
```
$ seq 3 | ./calc minus
-4      #1-2-3=4 
```
  - 乗算: `times`
  - 使用例:(N = 4)
```
$ seq 4 | ./calc times  
24      #1×2×3×4=24
```
  - 除算: `div`
  - 使用例:(N = 6)
```
$ seq 6 | ./calc div
0.001388888888888889      #1/2/3/4/5/6=1/720
```
#### 以下にエラーの起きる例を記載します
##### 未対応の演算子を指定した場合
```
$ seq 5 | ./calc add
unknown operation
```
- 対応していない演算子が指定された場合、エラーメッセージを表示して終了します。
##### 演算子を指定しなかった場合
```
$ seq 5 | ./calc
usage: calc [plus|minus|times|div]
```
- 演算子を指定しない場合、usage を表示して終了します。
