# 四則演算計算コマンド

## 概要
- このコマンドは標準入力から読んだ数値Nに対して四則演算を行うコマンドです。

## 機能概要
- 標準入力からの連続データを 1 行ずつ処理
- 整数・小数の両方に対応（int → float の自動で型変換されます。）
- 使用する演算はコマンドライン引数で指定
- minus / div でも自然な演算ができるよう、最初の値を初期値として扱う方式
- GitHub Actions による自動テストに対応

## 使用方法

### インストール(クローン)
```
git clone https://github.com/rinron777/robosys2025-2.git
```
- 上のコマンドを実行してください。

### 実行方法
```
seq N | ./calc (演算子)
```
#### (N)に任意の数、(演算子)に使用したい機能のコマンドを打ち込んで使用してください。
- 対応コマンド一覧(使用例付属)
  - 加算: `plus`
  - 使用例:(N = 5)
```
$ seq 5 | ./calc plus
15
```
  - 減算: `minus`
  - 使用例:(N = 5)
```
$ seq 5 | ./calc minus
-13
```
  - 乗算: `times`
  - 使用例:(N = 5)
```
$ seq 5 | ./calc times
120
```
  - 除算: `div`
  - 使用例:(N = 5)
```
$ seq 5 | ./calc div
0.008333333333333333
```
#### 以下にエラーの起きる例を記載します
##### 未対応の演算子を指定した場合
```
$ seq 5 | ./calc add
unknown operation
```
- 対応していない演算子が指定された場合、エラーメッセージを表示して終了します。
##### 演算子を指定しなかった場合
```
$ seq 5 | ./calc
usage: calc [plus|minus|times|div]
```
- 演算子を指定しない場合、usage を表示して終了します。
